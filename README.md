# Kashan Bites

Responsive landing page for a fictional food brand "Kashan Bites".

## Tech Stack
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (DOM manipulation)

## Features
- Mobile-friendly responsive design
- Smooth scrolling navigation
- Interactive mobile menu with toggle icon
- Header shadow effect on scroll
- Customer reviews and order form
